語音輸入面板

01 — API 金鑰設定

金鑰只存在這台電腦的瀏覽器本機儲存空間,不會上傳到任何第三方伺服器。電腦要借給別人用之前,記得按「清除金鑰」。

02 — 面板

這個瀏覽器不支援浮動小窗功能(Document Picture-in-Picture API)。請改用 Chrome 116 以上,或 Edge 116 以上版本;沒有這個功能,面板仍可在這個分頁內使用,但無法浮動置頂。
按 X 鍵:切換面板「浮動置頂」/「收回分頁內」。
按 Z 鍵:切換「開始/結束錄音」(面板需是作用中視窗,先點一下面板)。
錄音期間手可以拿麥克風,不用碰鍵盤,結束後結果自動複製到剪貼簿。
