# 語音輸入面板 Voice Input Panel

一個免安裝、純網頁的語音輸入工具。錄音後直接呼叫 Gemini API 完成「語音辨識 + 中英夾雜校正 + 標點符號潤飾」，結果自動複製到剪貼簿，可貼到任何軟體使用。

適合公司電腦等「不能安裝軟體」的環境：這是單一 HTML 檔案，用瀏覽器打開就能用。

## 功能特色

- **單一 Gemini API 金鑰**：語音辨識與文字潤飾一次呼叫完成，不用額外申請 Whisper/OpenAI 帳號
- **浮動置頂小窗**（Chrome / Edge 116+）：小窗可以浮在所有視窗最上層，切到 Word、Outlook、任何軟體都看得到
- **全程免滑鼠**：快捷鍵切換浮動/收回、開始/結束錄音，錄音期間雙手可以空出來拿麥克風
- **用量與費用顯示**：每次辨識後即時顯示這次用了多少 token、預估花費多少錢，並累計整個使用階段的總花費
- **金鑰只存在本機**：API 金鑰儲存在瀏覽器的 localStorage，不會上傳到任何第三方伺服器；電腦要借給別人用前，可以一鍵清除金鑰

## 使用方式

1. 用 Chrome 或 Edge（116 版以上）打開這個頁面（建議透過 GitHub Pages 的 `https://` 網址開啟，不要直接雙擊本機檔案，否則麥克風功能會被瀏覽器擋掉）
2. 貼上你的 [Gemini API Key](https://aistudio.google.com/apikey)，按「儲存金鑰」
3. 依照下方快捷鍵操作即可

### 快捷鍵

| 按鍵 | 功能 |
|---|---|
| `X` | 切換面板「浮動置頂」／「收回分頁內」 |
| `Z` | 切換「開始錄音」／「結束錄音」 |

> 快捷鍵需要面板所在的視窗（浮動小窗或瀏覽器分頁）是作用中視窗才會生效，先點一下面板即可。

### 基本流程

點一下面板 → 按 `X` 讓它浮動置頂 → 切到你要輸入文字的軟體 → 需要說話時點一下小窗、按 `Z` 開始錄音 → 手拿麥克風說話 → 再按一次 `Z` 結束 → 稍等一下辨識完成，結果自動進剪貼簿 → 切回原本軟體 `Ctrl+V`（Mac 是 `Cmd+V`）貼上

## 瀏覽器支援

| 功能 | Chrome / Edge 116+ | Safari |
|---|---|---|
| 錄音、辨識、潤飾、自動複製 | ✅ | ✅ |
| 浮動置頂小窗（`X` 鍵） | ✅ | ❌ 不支援，按了沒反應但不影響其他功能 |

Safari 使用者可以不用理會 `X` 鍵，面板固定顯示在分頁內，`Z` 鍵錄音功能一樣正常運作。

## 費用說明

計費模型為 Gemini 2.5 Flash，費率（每百萬 token，美元）：

- 音訊輸入：$1.00
- 文字輸入：$0.30
- 輸出：$2.50

以零星幾句話的日常使用量來說，每月費用大約落在幾毛到幾塊美金之間。小窗內會即時顯示每次的實際花費與累計金額。

## 隱私與安全

- API 金鑰只存放在你這台電腦瀏覽器的本機儲存空間（localStorage），不會傳送到任何第三方伺服器
- 錄音檔只會傳送到 Google 的 Gemini API 做辨識，不會經過其他中介伺服器
- 電腦要借給別人使用前，建議先按「清除金鑰」按鈕移除本機儲存的金鑰

## 已知限制

- 瀏覽器直接以 `file://` 開啟本機檔案時，麥克風功能會被瀏覽器基於安全性理由擋掉，請透過 GitHub Pages 或本機伺服器（`https://` 或 `http://localhost`）開啟
- 浮動小窗功能僅支援 Chrome / Edge 116 以上版本，Safari 與較舊版本瀏覽器不支援
